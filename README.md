# GPT-Trivia-UI

Frontend application of the trivia app leveraging OpenAI APIs to generate the trivia.

## Developing

```sh
npm install
npm run dev
```

## Tech stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Electron](https://www.electronjs.org/)
